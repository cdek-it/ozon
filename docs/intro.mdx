# Введение

Инструкция по настройке интеграции с магазина на Ozon с сервисом доставки от СДЭК

:::info

Данная интеграция предназначена для автоматического создания накладных СДЭК для заказов, оформленных по модели
RealFBS (Standard).
Подробнее про модель продаж RealFBS можно прочитать в [инструкции](https://seller.ozon.ru/media/beginners/realfbs-na-ozon-chto-takoe-i-kak-nachat/).

:::

Основные особенности интеграции:
- Автоматическое создание накладных СДЭК для заказов на Ozon без необходимости ручного переноса данных
- Синхронизация статуса доставки заказов между Ozon и СДЭК
- Прогнозирование размера упаковки на основе товаров заказа и подбор оптимального тарифа для отправки

<details>
    <summary>Принцип работы интеграции</summary>

    После настройки, система каждые 15 минут проверяет наличие новых заказов на Ozon.

    При обнаружении нового заказа, автоматически создается накладная в СДЭК.

    Накладная появляется в [личном кабинете](https://lk.cdek.ru), откуда ее можно распечатать и наклеить на посылку для
    передачи на доставку.
</details>

# Установка и настройка

Для работы интеграции, не требуется установка дополнительного ПО. Вся настройка происходит через личный кабинет СДЭК.

Для этого необходимо перейти на страницу [настроек интеграции](https://lk.cdek.ru/integration) и выбрать режим **DBS**.

<Image img={require('./img/start1.png')} />

Далее необходимо сгенерировать токен Ozon и ввести его в настройках интеграции.
Инструкцию по генерации токена можно прочитать на [отдельной странице](./ozon-key.mdx).
